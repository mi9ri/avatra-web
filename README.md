# Avatra Web

VRChat向けアバター改変構成管理サービスの個人開発プロジェクトです。

アバター改変時に使用したアイテム情報や構成情報を管理・共有できる仕組みを目指して開発しています。

## 概要

アバター改変時に以下の情報を整理・管理するためのWebサービスです。

* 使用アイテム管理
* 改変構成情報の管理
* JSON出力データとの連携
* Boothアイテム情報整理

※現在開発中

## 使用技術

* TypeScript
* Next.js
* React
* Supabase
* Vercel

## 開発環境

```bash
npm install
npm run dev
```
